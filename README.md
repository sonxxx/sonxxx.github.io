# Chrome Web 사이트

Javascript로 만든 간단한 크롬 클론 코딩.

- 닉네임으로 로그인을 하면 간단한 todo 리스트를 작성 할 수 있습니다.
- 위치 권한 허용 시 날씨API로 위치한 지역의 온도를 보여줍니다.
- 배경화면과 로그인 시 나오는 명언은 랜덤으로 보여줍니다.

> 사용 기술

- HTML, CSS

- Javascript
